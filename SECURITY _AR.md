
# سياسة الأمان – مشروع إتقان اللغة العربية

رغم أن المشروع تعليمي، إلا أن الدقة والأمان ما زالا مهمين.

## 1. الإبلاغ عن المشكلات
إذا وجدت:
- ترجمة غير صحيحة  
- خطأ لغوي  
- رابط لا يعمل  
- معلومة تعليمية مضللة  

يرجى فتح Issue بعلامة "Content-Security".

## 2. التحرير المسؤول
يجب التأكد من:
- صحة القواعد العربية  
- دقة الترجمة الإنجليزية  
- عدم وجود معلومات تاريخية أو ثقافية خاطئة  

## 3. أمان الملفات
يُسمح فقط برفع:
- ملفات PDF  
- ملفات Markdown  
- الصور التعليمية  

يُمنع رفع الملفات التنفيذية (.exe – .sh – .bat).
